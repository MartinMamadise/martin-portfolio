Traffic Engineering via Graph Theory & Linear Programming

PythonNetworkingGraph TheoryLinear ProgrammingOptimization
GitHub ↗

Network optimization system built around the NSFNET backbone topology that models traffic flow using graph theory and linear programming. Compares shortest-path routing (Dijkstra's) against Single-Commodity and Multi-Commodity Flow optimization to demonstrate real-world routing limitations at scale.

Stack NetworkX, PuLP + CBC Solver, Pandas, Matplotlib
Methods Dijkstra baseline → SCF cost minimization → MCF with Min-Max Utilization
MCF reduced maximum link utilization from over 600% down to under 75% — demonstrating traffic engineering strategies used in modern backbone networks.

Smart Agriculture Monitoring System

ESP32C/C++IoTFirebaseEmbedded
GitHub ↗

Multi-sensor irrigation system using Arduino and ESP32 that monitors soil moisture, ambient light, and reservoir levels in real time. Sensor data is transmitted over WiFi to a Firebase backend and mobile dashboard. An embedded control algorithm automatically regulates irrigation via a relay-controlled valve, while an I²C LCD provides on-site feedback.

Software C/C++ firmware, HTTP/REST to Firebase, TCP/IP on ESP32, MIT App Inventor
Hardware ESP32, capacitive soil sensor, LDR, HC-SR04 ultrasonic, relay module, I²C LCD
Real-time monitoring with remote access — sensor readings at short intervals enabled timely detection of environmental changes and more efficient water use.
02

Solar-Powered Washing Machine Controller

ATmegaC++SolarEmbeddedEnergy Management
GitHub ↗

Embedded controller that runs a washing machine entirely on solar power. Continuously monitors energy availability and dynamically adjusts wash cycles — pausing and resuming as solar conditions change, preventing mid-cycle failures. Designed for South Africa's load-shedding environment. Manages all stages (fill, wash, rinse, drain, spin) autonomously without grid dependency.

Software Embedded C/C++ (Arduino IDE), Tinkercad simulation, IRremote library
Hardware Arduino Uno (ATmega328P), 50W solar panel, 12V battery, L298N motor driver, DC pump, solenoid valve
Full off-grid operation under optimal conditions. Intelligent power-aware control eliminates mid-cycle failures and resumes automatically — a scalable approach to renewable-powered appliances.
